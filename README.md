# Commit_

A minimalist, developer-focused habit, task, and note tracker. 

I built this because I wanted something simple, dark-themed (OLED black, of course), and distraction-free for tracking my daily commits—whether that's writing code, reading, or just getting things done.

## Features
- **Habits**: Track your daily streaks.
- **Tasks**: A simple to-do list.
- **Notes**: Jot down thoughts or quick snippets.
- **Career**: Keep an eye on the bigger picture.

## Setup
Just a simple web app wrapped with Capacitor for Android. 

```bash
npm install
npm run build
npx cap sync android
```
